# bot
bot for everything

##Prerequisites

### node first
### redis second

##Install

```shell
git clone https://github.com/nutbox-dao/nutbox-faucet.git
cd nutbox-faucet/bot/discord
npm install discord.io winston -save
npm install https://github.com/woor/discord.io/tarball/gateway_v6
node bot
```
